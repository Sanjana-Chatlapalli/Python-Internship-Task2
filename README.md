# Task 2: Variables, Data Types & Type Conversion

## Objective
This task demonstrates basic Python concepts:
- Variables
- Data types (Integer, Float, String, Boolean)
- Type checking
- Type conversion
- Arithmetic operations
- Error handling
- Dynamic typing

## Python Code
The program is saved in `datatypes_demo.py` and demonstrates all the above concepts.

**Code snippet:**
```python
age = 20
height = 5.6
name = "Sanjana"
is_student = True

print("Age:", age)
print("Height:", height)
print("Name:", name)
print("Is Student:", is_student)
# Arithmetic operations
a = 10
b = 3
print("Addition:", a + b)
print("Division:", a / b)
# Type conversion
num = int(input("Enter a number: "))
print("Number + 5 =", num + 5)
```
**How to Run**
Open terminal or command prompt
Navigate to the folder containing datatypes_demo.py

**Run the program:**
python datatypes_demo.py

**Expected Output**
Age: 20
Height: 5.6
Name: Sanjana
Is Student: True
--- Arithmetic Operations ---
Addition: 13
Division: 3.3333333333333335
--- Type Conversion ---
Enter a number: 10
Number + 5 = 15
